# Trading-212-Accounting
Έφτιαξα ένα δωρεάν HTML εργαλείο που διαβάζει τα CSV exports του Trading 212 και τα οργανώνει σε μορφή που μπορεί να χρησιμοποιήσει απευθείας ένας λογιστής.

Το πρόβλημα: Κατεβάζεις το CSV από το T212, είναι ένα χάος — αγορές, πωλήσεις, τόκοι, κινήσεις κάρτας, cashback, καταθέσεις, όλα μαζί σε ένα αρχείο. Ο λογιστής σου θέλει να ξέρει τι αγόρασες, τι πούλησες, τι κέρδος/ζημία έκανες, πόσους τόκους πήρες, και τι χρεώσεις έγιναν — ξεχωριστά και καθαρά.

Τι κάνει:

• Επισκόπηση — σύνολα αγορών, πωλήσεων, P&L, τόκων, καταθέσεων, δαπανών κάρτας, cashback, FX fees. Ανάλυση ανά ticker (ISIN, μετοχές, κέρδος/ζημία) και μηνιαία ανάλυση.

• Μετοχές/ETFs — κάθε buy/sell με φίλτρα για ticker, ημερομηνία, ποσό, ISIN. Live summary cards.

• Τόκοι — ξεχωριστή ενότητα, ημερήσιες εγγραφές, μηνιαία σύνολα, μέσοι όροι.

• Κινήσεις κάρτας — φίλτρα ανά έμπορο, κατηγορία, μήνα. Breakdown ανά έμπορο και κατηγορία με ποσοστά.

• Καταθέσεις & Cashback — ξεχωριστές ενότητες.

• Raw data — όλα τα δεδομένα με free-text search.

Όλοι οι πίνακες κάνουν sort, όλα τα sections έχουν CSV export, τα φίλτρα είναι live. Τρέχει 100% τοπικά στον browser — δεν ανεβαίνουν δεδομένα πουθενά.

Απλά κατεβάζεις το HTML, το ανοίγεις, σύρεις το CSV μέσα, και είσαι έτοιμος. Στείλε τα exported CSVs στον λογιστή σου ή κάτσε μαζί του με το εργαλείο ανοιχτό.
